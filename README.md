# DigiTools-dev 🛠️

A modern, high-performance web application featuring a suite of digital tools designed for efficiency. This project was built using Replit AI and is optimized for deployment on Netlify.

## 🚀 Live Demo
Check out the live application here:  
**[https://digitools-dev.netlify.app/](https://digitools.netlify.app/)**

## ✨ Features
- **Modern UI/UX:** Built with Tailwind CSS for a sleek, responsive design.
- **Fast Performance:** Powered by Vite and React for a seamless user experience.
- **Robust Backend:** Uses a Node.js server with Drizzle ORM for efficient data management.
- **Type Safety:** Entirely written in TypeScript to ensure code quality and reliability.

## 🛠️ Tech Stack
- **Frontend:** React, Tailwind CSS, Lucide React (Icons), Shadcn UI
- **Backend:** Node.js, Express
- **Database:** Drizzle ORM (PostgreSQL ready)
- **Build Tool:** Vite
- **Deployment:** Netlify

## 📁 Project Structure
- `/client`: Frontend React application.
- `/server`: Backend Express server and API logic.
- `/shared`: Shared TypeScript types and schemas.
- `/attached_assets`: Static assets and media files.

## ⚙️ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/TechGenDM/DigiTools-dev.git](https://github.com/TechGenDM/DigiTools-dev.git)
   cd DigiTools-dev
2. Install dependencies:
   ```bash
   npm install
Running Locally
1. Start the development server:
   ```bash
   npm run dev
2. Open your browser and navigate to http://localhost:5000.

### 📦 Deployment
This project is configured for easy deployment on Netlify.
The netlify.toml file handles the build settings and redirects.
Ensure your environment variables are configured in the Netlify dashboard for production.
